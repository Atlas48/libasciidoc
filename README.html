This project is an attemtp to implement the famous Asciidoctor project in Golang to make it natively available for other Golang projects.

Supported syntax

  • ✓ Titles / headings

  • ❏ Quoted text

    • ✓ Bold

    • ❏ Italic

  • ❏ Lists

    • ✓ Unordered, simple list using the * marker

    • ❏ Ordered

  • ❏ Links and images

    • ✓ external links (https://, http://, ftp://, irc://, mailto:)

    • ❏ inline images

    • ✓ block images

Supported backend/output

  • ❏ HTML5
